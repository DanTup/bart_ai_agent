# Bart - A Simple AI Agent

## Overview

A simple AI agent built in Dart that. Works via the terminal or through a simple web app.

## Basic Usage

- `dart bin/main.dart` - run at the terminal
- `dart bin/main.dart --web` - run via a simple web app
